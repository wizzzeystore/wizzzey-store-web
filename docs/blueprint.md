# **App Name**: Tailored

## Core Features:

- Home Page: Home page showcasing featured products and categories.
- Product Card Component: Product cards displaying images, names, and prices, using a rectangle borderless design.
- Shop Page with Filters: Shop page listing all clothing items with filter panel (price range, categories, sorting).
- Profile Page: User profile page to manage account details and order history.
- Cart Page: Cart page to view selected items and proceed to checkout.
- Orders Page: Orders page displaying past order history.
- Order Confirmation Modal: Order placement confirmation modal with animation after successful 'create order' API call. The confirmation tool guides the customer to their orders page.

## Style Guidelines:

- Primary color: HSL(210, 60%, 50%) - RGB(51, 153, 255) - A vibrant blue, reflecting trust and reliability.
- Background color: HSL(210, 20%, 95%) - RGB(242, 247, 255) - A light, desaturated blue to provide a clean backdrop.
- Accent color: HSL(180, 50%, 50%) - RGB(64, 191, 191) - A contrasting cyan to highlight interactive elements and calls to action.
- Body and headline font: 'Inter' (sans-serif) for a modern, neutral look that ensures readability and a clean aesthetic.
- Code font: 'Source Code Pro' (monospace) for displaying code snippets.
- Use a grid-based layout for product listings to maintain a structured and visually appealing arrangement.
- Subtle transitions and animations on button hovers and page transitions to enhance user experience.
